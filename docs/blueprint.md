# **App Name**: Carwash Manager

## Core Features:

- Counter Agent Management: Manage counter agents, including their name, list of cars (license plate number), and company owner(s). Support for multiple companies owned by a single owner.
- Aggregator Management: Manage aggregators, including their list of cars and price list.
- Cash Client Tracking: Track cash, card, and transfer payments from direct clients.
- Dashboard: Provide a dashboard summarizing all key metrics (revenue by source, total washes, etc.).
- AI Report Generation: Generate reports on key metrics, suggesting improvements or identifying potential fraud using an AI-powered tool. It is going to output trends in performance, etc.

## Style Guidelines:

- Primary color: Moderate blue (#64B5F6) to evoke trust and cleanliness, without being clich√©. This blue is versatile and contrasts well with light backgrounds.
- Background color: Light blue-gray (#F0F4F8), a desaturated shade of the primary blue, creates a clean, professional backdrop.
- Accent color: Slightly purple blue (#5C6BC0), contrasting in brightness and saturation with the primary, to highlight key actions and data points.
- Headline font: 'Poppins', a geometric sans-serif, gives a modern and fashionable feel. Body font: 'Inter', a grotesque-style sans-serif, with a modern, machined, objective, neutral look, will work well for the app body. It has clear legibility.
- Use a set of modern, outline-style icons that represent different data points and actions within the app. These icons should be easily recognizable and complement the overall design.
- A clean and intuitive layout with a focus on usability. Key information should be easily accessible. Use clear visual hierarchy to guide the user through the app.
- Subtle transitions and animations to enhance user experience. For example, loading animations, confirmation animations, and interactive hover effects.
